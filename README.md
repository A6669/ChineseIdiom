# 成語典電子書
### 如何利用成語典學習文言文閱讀

許多人學習古文往往流於「為學而學」，但使用《成語典》則是「為了求知而讀」，兩者的心態截然不同。以下是具體的操作建議：

1.  **將「典源」當作精讀素材** 《成語典》中提供的「典源」，通常是該成語最早的出處。這些文言段落篇幅短小、難易適中，非常適合利用零碎時間進行閱讀，閱讀時建議使用蘋果的Books閱讀器，其他epub閱讀器可能顯示異常。
2.  **利用「書證」累積語感** 在「書證」欄位中，辭典會列舉該成語在不同朝代、不同古籍中的應用實例。 你可以試著對照不同朝代的用法。例如同一個成語，在《史記》裡是如何使用的？到了《紅樓夢》裡又是什麼樣貌？這能大幅豐富漢語語感。

**簡單來說，這部成語典是用來「通讀」的，而不僅是為了「查閱」而存在的。**

更具體可以參考[如何利用成語典學習文言文閱讀](https://zhuanlan.zhihu.com/p/1999819491555710542)

### 成語典序
====

蓋聞言之精者，莫若成語。夫成語者，其辭約，而其旨尤遠，其言近，而其義甚玄。後世學者，苟不明其源，未究其本，則難窺成語之閫奧，亦無以彰文辭之光華。 今之成語典者，捃摭成語千有餘條，編次有序，燦然大備。考其體例，非特釋義而已，抑亦條分縷析，探賾索隱，博引載籍。凡典源之本末，靡不審證，讀者覽之，如指諸掌矣。 夫欲習古文者，恆病無入道之法。繇成語以登古文之階，不亦善乎？讀其文，則知先賢之行事。味其語，則通古今之辭章。繇是日日新又日新，沈潛涵泳，深翫其閒，則非惟成語喻於心，漢魏之風亦有所得焉。 蓋觀斯編也，能沾丐後學以追古人之高致，克惠及來者以成今人之雅尚。夫誠如是，則紹文治賡教化者，固將有乎爾，則亦有乎爾，是為序。

知乎 無雙

西元二千零二十六年一月三十日
